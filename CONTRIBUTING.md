# Contributing

This repo is developed with Spec Kit: constitution → spec → plan → tasks → implement.

1. Read [`.specify/memory/constitution.md`](.specify/memory/constitution.md).
2. Follow [docs/spec-driven-development.md](docs/spec-driven-development.md).
3. Keep `yarn test:smoke` green; add or extend tagged tests for risk you touch (see [tests/README.md](tests/README.md)).
4. Optional demo stack: `docker compose up --build` → `http://127.0.0.1:8080`.

Agent skills for the Spec Kit loop live under [`.cursor/skills/`](.cursor/skills/).
OpenAPI starter: [docs/openapi.yaml](docs/openapi.yaml).
